Business rules
    UX notes
    Outcome layar

    Request tercatat rapi, status jelas, dan potensi bottleneck terlihat.

    Usulan information architecture

    Untuk memudahkan implementasi, prototype ini juga memetakan halaman inti dan kepemilikan perannya.

    Rute halaman inti

    Semua route di bawah adalah usulan struktur URL untuk memisahkan peran Admin Pusat dan Admin Pengelola.

    /gov-onboarding/requests
    Admin Pusat

    Daftar seluruh request, filter status, SLA, dan aksi cepat.

    /gov-onboarding/requests/:id
    Admin Pusat

    Detail surat, data admin, warning duplikasi, dan control invitation.

    /gov-onboarding/invitations/:token/profile
    Admin Pengelola

    Form profil instansi, upload logo, draft, dan submit.

    /gov-onboarding/verifications/:id
    Admin Pusat

    Checklist review, keputusan valid/revisi/reject, dan audit trail.

    /employer/dashboard
    Instansi Active

    Reuse dashboard employer untuk posting lowongan, leads, dan talent match.

    /gov-onboarding/monitoring
    Admin Pusat

    Monitoring operasional, histori email, resend rate, dan audit log.

    Prinsip keputusan UI

    Status-first interface

    Pengguna harus selalu tahu request sedang berada di status apa, siapa aktor berikutnya, dan apa aksi yang dibutuhkan.

    Exception visible by default

    Expired link, revisi, dan duplikasi tidak boleh tersembunyi di log; semua harus tampak pada permukaan UI.

    Reuse over rebuild

    Onboarding khusus pemerintah berakhir di dashboard employer existing, sehingga scope tetap fokus dan realistis.

    Audit as a product feature

    Jejak email, perubahan status, dan keputusan reviewer bukan aksesoris operasional, tetapi bagian inti pengalaman.

    Lampiran workflow acuan

    Gambar di bawah disematkan agar reviewer bisa membandingkan prototype dengan flow asli yang menjadi dasar BRD.

    Lihat workflow acuan
    Prototype konseptual ini disiapkan untuk diskusi business, product, dan delivery. Warna, copy, dan route masih dapat disesuaikan pada tahap design refinement.
